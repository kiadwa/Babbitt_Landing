## Getting Started

### Local Development

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (with http-server)
   npx http-server
   ```

### Deploy to GitHub Pages

1. Create a new repository on GitHub (or use this one)
2. Push all files to your repository
3. Go to your repository settings
4. Navigate to **Pages** in the left sidebar
5. Under **Source**, select the branch (usually `main`) and folder (`/root`)
6. Click **Save**
7. Your site will be available at `https://[username].github.io/[repository-name]`

### Customization

- **Colors**: Edit the CSS variables in `styles.css` (lines 6-12)
- **Content**: Update the text in `index.html`
- **Features**: Modify the feature cards in the Features section
- **Contact**: Update email addresses and links in the Contact section
